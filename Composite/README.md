# Composite
## 概要
容器と中身を同一して、再帰的な構造を作る実装パターン。
ファイルシステムのディレクトリのようなイメージ。ディレクトリの中にファイルとディレクトリ(フォルダ)が存在し、そのディレクトリの内部にもファイルとディレクトリが存在している。このような入れ子になった構造を実装する。
